# 🎯 Lead Scoring using Machine Learning

A Machine Learning project developed as part of the **Celebal Technologies Excellence Internship 2026**.

## 📌 Project Overview

X Education generates a large number of leads through various marketing channels, but only around **30%** of these leads convert into customers.

This project builds a Machine Learning model to assign a **Lead Score (0–100)** to every lead, helping the sales team prioritize high-potential customers and improve conversion rates.

---

## 🎯 Objectives

- Analyze customer lead data
- Perform data cleaning and preprocessing
- Explore the dataset using EDA
- Build multiple Machine Learning models
- Compare model performance
- Predict lead conversion probability
- Generate Lead Scores (0–100)
- Categorize leads into:
  - 🔥 Hot Lead
  - 🟡 Warm Lead
  - 🔵 Cold Lead

---
